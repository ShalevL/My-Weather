React | Redux Toolkit. 

Design with Material UI & React Bootstrap.

installation instructions
1. download code
2. navigate to folder
3. init a valid AccuWeather API_KEY in store/index.js (my dafault API_KEY already initiated)
4. run npm install
5. run npm start
